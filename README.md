A professional graphical interface for Lineage 2 server administration. Complete management of players, skills, items, and server configurations.

## ✨ Features

### 👥 Player Management
- ✅ List all online/offline players
- ✅ View complete account and character details
- ✅ Edit levels, experience, class, race
- ✅ Manage inventory and equipment
- ✅ Reset skills and stats
- ✅ Ban/Unban players
- ✅ Send in-game messages and notifications

### 📊 Skill Management
- ✅ View all server skills
- ✅ Edit skill attributes (damage, cooldown, MP consumption)
- ✅ Create new skills
- ✅ Clone existing skills
- ✅ Import/Export skill XML files

### ⚔️ Item Management
- ✅ List all items
- ✅ Create and edit items
- ✅ Manage monster drops
- ✅ Control NPC shops
- ✅ Send items to players

### 🌍 World Management
- ✅ Server status (uptime, online players, etc)
- ✅ Raid and boss management
- ✅ Event control
- ✅ Global announcements
- ✅ Real-time server logs

### ⚙️ Configuration
- ✅ Edit server settings (rates, exp multiplier, etc)
- ✅ Database management
- ✅ Backup and restore
- ✅ Plugins and mods support
